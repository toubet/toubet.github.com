MarkDown语法

发表于 2019-10-10 更新于 2020-03-06 分类于 Markdown 阅读次数：

1.基本语法

1.1.分级标题

第一种写法
    # 这是一级标题
    ## 这是二级标题
    ### 这是三级标题
    #### 这是四级标题
    ##### 这是五级标题   
    ###### 这是六级标题
第二种写法
    这是一个一级标题
    ===================

    这是一个二级标题
    -------------------

效果如下：

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

1.2.字体

- 加粗： **加粗**   __加粗__
- 斜体： *斜体*     _斜体_
- 斜体加粗： ***斜体加粗***    ___斜体加粗___  
- 删除线： ~~删除线~~
  • 效果:
    • 加粗 加粗
    • 斜体 斜体
    • 斜体加粗 斜体加粗
    • 删除线

1.3.链接

Markdown 支持两种形式的超链接语法： 行内式和参考式两种形式，行内式一般使用较多.

  • 行内式：

    • 效果1:跳转到菜鸟教程
    • 效果2：跳转到菜鸟教程

       参考式超链接一般用在学术论文上面，或者另一种情况，如果某一个链接在文章中多处使用,那么使用引用的方式创建链接将非常好，它可以让你对链接进行统一的管理参考式链接分为两部分，文中的写法 [链接文字][链接标记]，在文本的任意位置添加[链接标记]:链接地址 “链接标题”，链接地址与链接标题前有一个空格。

  • 参考式：

    • 几个网站GitHub、知乎以及简书，菜鸟教程是一个不错的学习网站。

  • 自动链接

  • 效果：

    • http://example.com/
    • address@example.com

       网页中，锚点其实就是页内超链接，也就是链接本文档内部的某些元素，实现当前页面中的跳转。在你准备跳转到的指定标题后插入锚点{#标记}，然后在文档的其它地方写上连接到锚点的链接。

1.4.分割线

       可以在一行中用三个以上的星号(*)、减号(-)、底线(_)来建立一个分隔线，行内不能有其他东西。也可以在星号或是减号中间插入空格。

1.5.代码块

       插入程序代码的方式有两种，一种是利用缩进(tab), 另一种是利用六个英文“ ` ”包裹.

1.6.引用

    在被引用的文本前加上>符号就可以了。
> 文本
>>文本
>>>文本

1.7.列表

无序列表：
使用 *，+，- 表示无序列表。
注意：符号后面一定要有一个空格，起到缩进的作用。

- 列表文字          
- 列表文字

+ 列表文字
+ 列表文字

* 列表文字
* 列表文字

效果：

  • 列表文字
  • 列表文字
  • 列表文字
  • 列表文字
  • 列表文字
  • 列表文字

       有序列表：使用数字和一个英文句点表示有序列表。
注意：英文句点后面一定要有一个空格，起到缩进的作用。

1.8.表格

表格写法：

姓名|学号|班级
:-:|:-:|:-:
小红|01|1
小明|02|1

       表格对齐方式：指定表格单元格的对齐方式，冒号在左边表示左对齐，右边表示有对齐，两边都有表示居中。

姓名 学号 班级
小红 01 1
小明 02 1
